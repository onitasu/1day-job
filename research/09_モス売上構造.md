## 3. 全店売上高と連結売上高の関係

### 差の構造

```
全店売上高（推定 約1,300億円）= 直営全店 + FC全店の店頭売上合計

連結売上高（約962億円）の構成:
  ├── 直営店売上高: 241億円（直営40店の飲食売上そのもの）
  ├── FC向け食材卸売: 491億円（FC店が本部から仕入れる食材代）
  ├── ロイヤリティ等: 29億円（FC売上の2%）
  ├── 海外事業: 166億円
  └── その他: 35億円

差額（約340億円）= FC店の店頭売上のうち食材仕入れ以外の部分
                   （FC店側の人件費・賃料・オーナー利益等）
```

### 図解: FC加盟店の売上の流れ

```
FC加盟店の店頭売上（推定 約1,060億円）
  ├── → モス本部への食材仕入代金: ~491億円 → 連結売上に計上
  ├── → ロイヤリティ+広告費（売上の2%）: ~21億円 → 連結売上に計上
  └── → FC店自身の費用・利益: ~548億円 → 連結売上に含まれない
```

## 5. 戦略的示唆

### モスの売上構造から見える垂直統合の機会

1. **食材卸売が収益の柱（491億円）**: これは既に「製造→供給」の垂直統合が部分的に機能していることを意味する。紅梅食品工業（22%出資）を完全子会社化すれば、**製造マージンもグループ内に取り込める**

2. **低ロイヤリティモデルの裏返し**: ロイヤリティ1%は安いが、食材マージンで回収している。つまり**食材の付加価値を上げる（=川上統合を深める）ほど、FC向け卸売の利益率が上がる構造**

3. **直営40店 vs FC1,267店**: FC比率97%は業界最高水準。これ以上のFC化余地は少ないが、逆に言えば**食材供給を通じたFC店へのグリップが強い**

4. **冷凍食品リテール展開**: 現在のEC（Life with MOS）は小規模だが、製造機能を内製化すれば**B2C冷凍食品事業を新セグメントとして立ち上げる**余地がある
