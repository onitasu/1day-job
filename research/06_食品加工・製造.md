# 食品加工・製造の実態と関連市場

## 1. パティ（ハンバーグ）の製造体制: 外注

### 製造委託先: 紅梅食品工業株式会社
- 1972年のモスバーガー創業時からパティとミートソースの受託生産を担当
- **モスフードサービスが株式22.20%を保有**（持分法適用関連会社）

### 紅梅食品工業の概要
| 項目 | 内容 |
|---|---|
| 本社 | 東京都練馬区 |
| 設立 | 1953年3月（創業1948年） |
| 製造拠点 | 栃木県日光市に3工場 |
| 第一工場 | パティ製造 |
| 第二工場 | ソース・惣菜製造 |
| 第三工場 | 生麺製造 |
| 主要販売先 | モスフードサービス、日本アクセス等 |
| 品質管理 | ISO22000準拠、2024年3月にJFS-B規格適合証明取得 |

> 出典: [WF-NET モスバーガーパティ工場視察](https://frozenfoodpress.com/2018/06/11/mos-burger-meat-patty-factory-kohbai/)、[紅梅食品工業 会社案内](http://www.kohbai.co.jp/company.html)、[irbank - 紅梅食品工業との資本関係](https://irbank.net/E02675/ac?m=%E7%B4%85%E6%A2%85%E9%A3%9F%E5%93%81%E5%B7%A5%E6%A5%AD%E6%A0%AA%E5%BC%8F%E4%BC%9A%E7%A4%BE)

## 2. バンズ（パン）の製造体制: 外注

### 製造委託先: 山崎製パン（現在はほぼ集約）
- もともとは地域別に複数メーカーが製造:
  - 北海道: 日糧製パン
  - 関東: 山崎製パン
  - 九州: キムラヤ
  - 中国地方: リョーユーパン
  - 沖縄: 具志堅パン
- **2017年7月のバンズ刷新を機に山崎製パンへの集約が進行**
- 新バンズは全粒粉配合、発酵種「ルヴァン」使用

## 戦略的示唆

1. **紅梅食品工業がサプライチェーンの要**: パティとソース類を1社に集約。22.20%の出資で一定の影響力を保持するが、依存度は非常に高い
2. **ファブレスモデルの長所と限界**: 自社で製造設備を持たないため資本効率は高いが、品質・コスト・供給の主導権が限定的
3. **紅梅食品工業の完全子会社化**: 現在22.20%出資 → 過半数取得 or 完全子会社化で、製造機能をグループ内に取り込む選択肢がある
4. **パティの内製化/加工品のリテール展開**: 製造機能を自社化できれば、店舗向けだけでなく冷凍食品としてのB2C/B2B販売が可能に
