# 補足資料 06：戦略提言 — M&A候補・市場分析の詳細

## 1. 4つの顧客セグメントの市場分析

### セグメント①: 自社FC＋買収ブランド（製造内製化）

**目的**: 現在外部に流出している製造マージンをグループ内に取り込む

| 項目 | 現状 | 転換後 |
|---|---|---|
| パティ製造 | 紅梅食品に外注 | **グループ内製造** |
| ソース製造 | 紅梅食品に外注 | **グループ内製造** |
| バンズ製造 | 山崎製パンに外注 | **グループ内製造** |
| 製造マージン | 外部に流出 | **グループ内に留保** |

**参照: Domino's SC事業**
- FC店への食材・機器・消耗品の供給で売上$2.85B（全体の60.5%）
- モスの食材卸売491億円（51%）は既にDomino's型の構造に近い
- 製造を内製化すれば、このマージンが拡大する

### セグメント②: 中堅外食チェーン（50-300店）向けB2B

**市場規模**: ~3,000億円（潜在）

**中堅チェーンのペインポイント**:
1. スケールメリットの欠如（大手のような調達コスト優位なし）
2. 人手不足で店舗での仕込みが困難化
3. 品質均一化の難しさ（店舗ごとのバラつき）
4. 自前CK投資は数億〜数十億円で過大

**競合状況**:
| プレイヤー | 内容 | 規模 |
|---|---|---|
| ゼンショーGFF | グループ内製造が主、外販は副次的 | その他セグメント4,130億円 |
| シコメルフードテック | 飲食店のレシピを預かり仕込み済み食材を納品 | スタートアップ |
| DDグループ共同購買PF | 大手の購買力を中小に開放 | 限定的 |

**核心**: 「外食チェーンがB2B食材サプライヤーになった事例」は国内ほぼ空白。**先行者利益のチャンス**。

### セグメント③: スーパー/コンビニ向けリテール

**市場規模**:
| カテゴリ | 市場規模 | 成長率 |
|---|---|---|
| 冷凍食品（家庭用） | 4,062億円 | +4.4%/年 |
| 冷凍食品（業務用） | 8,005億円 | — |
| ソース・調味料 | 5,770億円 | CAGR 6.45% |

**参照: イートアンドフーズ（大阪王将）**
- 外食チェーン→冷凍食品メーカーへの転換に成功
- 食品事業売上199億円
- 冷凍餃子を量販店・コンビニに販売

**モスの差別化要素**:
- 「モス」ブランドの消費者認知度（50年の歴史）
- 独自レシピ（ミートソース13回改良、累計13億個）
- 「安心・品質・国産」のブランドイメージ

### セグメント④: 業務用野菜卸売

**市場規模**: 外食産業の食材仕入れ総額~10-11兆円のうち、野菜卸売は数千億円規模

**デリカフーズHDの現状** (M&A候補):
| 項目 | データ |
|---|---|
| 売上高 | 588億円 |
| 時価総額 | 82億円 |
| PBR | **0.6倍（割安）** |
| 顧客数 | 外食チェーン多数 |
| 機能 | カット野菜加工+全国配送 |

→ デリカフーズの既存売上588億円がベースラインを保証。成長投資により700〜1,000億円への拡大を見込む。

## 2. M&A候補の詳細プロファイル

### 1st: 紅梅食品工業（残77.8%取得）

| 項目 | 内容 |
|---|---|
| **取得コスト** | **25〜40億円** |
| 推定売上 | 60〜80億円 |
| 純利益 | 2.21億円 |
| 総資産 | 111億円 |
| 利益剰余金 | 29.5億円 |
| 工場 | 栃木県日光市に3工場（パティ/ソース/麺） |
| 取引歴 | 53年（1972年創業時から） |
| **獲得する能力** | **パティ・ソース・麺の製造内製化** |

**なぜ1stか**:
- 既に22%出資、53年の取引関係で統合リスク最小
- 投資余力400億円の9%と小規模
- 成功体験を積んでから次のM&Aに進める
- 製造マージンの即時取り込みが可能

### 2nd: デリカフーズHD

| 項目 | 内容 |
|---|---|
| **取得コスト** | **110〜140億円** |
| 上場市場 | 東証スタンダード（3392） |
| 売上高 | 588億円 |
| 時価総額 | 82億円 |
| PBR | 0.6倍 |
| 従業員 | ~1,800名 |
| 拠点 | 全国に加工・配送拠点 |
| **獲得する能力** | **外食向け野菜SC（売上588億＋成長投資）** |

**なぜ2ndか**:
- PBR 0.6倍で**バリュエーション上割安**
- 既存顧客基盤（外食チェーン多数）がB2B外販のベースライン
- モスの農業基盤（契約農家2,379軒）との統合シナジー
- 野菜SCは温度管理・物流が参入障壁 → 模倣されにくい

### 3rd: 製パン会社（第一屋製パン or 日糧製パン）

| 候補 | 売上 | 時価総額 | 推定コスト | 特徴 |
|---|---|---|---|---|
| **第一屋製パン** | 272億円 | 42億円 | 55〜70億円 | 関東圏基盤、利益改善中 |
| **日糧製パン** | 190億円 | 46億円 | 60〜80億円 | 北海道基盤、山崎製パンが筆頭株主 |

**獲得する能力**: バンズの内製化。現在山崎製パンに完全外注しているバンズを自社生産に切り替え。

### 4th: 石井食品

| 項目 | 内容 |
|---|---|
| **取得コスト** | **70〜90億円** |
| 上場市場 | 東証スタンダード（2894） |
| 売上高 | 95億円 |
| 時価総額 | 55億円 |
| 主力商品 | ハンバーグ・ミートボール（「イシイのおべんとクン」） |
| **獲得する能力** | **ハンバーグ加工能力 + リテールチャネル** |

**なぜ4thか**:
- ハンバーグ製造のノウハウがモスのパティ事業と直結
- 既にスーパー向けの販売チャネルを保有 → リテール展開の足がかり
- 「イシイ」ブランド＋「モス」ブランドのダブルブランド展開も可能

## 4. 構造転換の詳細図解

```
【現状のバリューチェーン】

契約農家(2,379軒) ──→ 紅梅食品(製造/外部) ──→ モス本部(卸売491億) ──→ FC店(1,307店)
モスファーム(10拠点)     山崎製パン(バンズ/外部)    3PL(物流/外部)
                         ↑マージン外部流出          ↑物流費外部支払

収益化: FC卸売491億 + 直営241億 + ロイヤリティ29億 = 761億（国内）

【転換後のバリューチェーン】

契約農家 ──→ モス製造（紅梅+パン+石井） ──→ モスSC本部 ──→ ① 自社FC+買収ブランド
モスファーム     ↓(余剰キャパシティ)              ↓               ② 中堅チェーンへOEM
                 外部B2B供給                       ↓               ③ スーパー/コンビニ冷凍食品
デリカフーズ ──→ 野菜加工・配送 ──────────→ ④ 外食チェーン全般へ野菜卸売

収益化: 製造マージン + FC卸売 + B2B外販 + リテール + 野菜卸売 = 多層収益
```
