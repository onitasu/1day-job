# Slide 03：既存モデルの天井

## メインメッセージ
> **店舗ビジネスの全レバーを楽観的に積み上げても、目標の62%（6,710億円）が上限。残り4,100億円は「店舗の外」で作るしかない。**

---

## 店舗ビジネスの成長レバーを網羅的に検証

### 店舗数の天井（現状1,307店）

| レバー | 方法 | ベンチマーク | 10年後の寄与 |
|---|---|---|---|
| FC出店加速 | オーナー獲得強化 | コメダ年+70（モスは現状+8） | +350〜1,000店 |
| 新フォーマット | 小型店・ゴースト | 日販は従来型の1/3〜1/2 | +40〜200店相当 |
| M&A（外食） | W-FK+フレッシュネス | 108+157＝265店 | +108〜265店 |
| **合計** | | | **1,835〜2,832店** |

### 1店舗あたり売上の天井（現状0.99億円）

| レバー | 方法 | ベンチマーク | 寄与 |
|---|---|---|---|
| 朝食帯 | モーニング導入 | マクドナルド朝食＝日販の20-25% | +0.18〜0.30億 |
| デリバリー | 全店展開 | マクドナルド純増分≒0.19億/店 | +0.05〜0.10億 |
| DT・デジタル・速度 | 各施策 | DT店は+20-30%の日販 | +0.10〜0.21億 |
| 価格・セット改善 | 値上げ+セット率 | CPI年+2.5%×10年 | +0.24〜0.35億 |
| マルチブランド・高単価化 | W-FK等の高単価ブランド追加＋メニュープレミアム化 | バーガーキング~1.3億/店、マクドナルド2.94億が業界上限 | +0.05〜0.10億（ポートフォリオ混合効果） |
| **合計** | | | **1.61〜2.05億/店** |

※ マルチブランド化はポートフォリオ内の客単価分布を改善するが、日本のファストフード1店あたり売上の構造的上限（マクドナルドの2.94億円）は超えられない。

### 天井の統合

| シナリオ | 国内 | 海外 | **合計** | **目標比** |
|---|---|---|---|---|
| **現実的** | 2,960億 | 300億 | **3,260億** | **30%** |
| **楽観** | 4,560億 | 550億 | **5,110億** | **47%** |
| **超楽観** | 5,810億 | 900億 | **6,710億** | **62%** |

**→ マルチブランド化・高単価化を含む全レバーが超楽観で着地しても、不足額は4,100億円。「店舗ビジネスの中でもっと頑張る」では届かない構造的な壁がある。**
