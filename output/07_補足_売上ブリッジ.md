# 補足資料 07：売上ブリッジ — 各ラインアイテムの前提と感応度分析

## 1. A. 外食事業の積み上げ根拠

### A-1. 国内モスバーガー（既存業態最大化）

| 前提 | Base | Upside |
|---|---|---|
| 店舗数 | 1,835店 | 2,832店 |
| 1店あたりSWS | 1.56億円 | 1.69億円 |
| **合計** | **2,860億円** | **4,780億円** |

前提の根拠:
- **店舗数Base**: FC年+35店 × 10年 + M&A(W-FK108店) + 新フォーマット40店
- **店舗数Upside**: FC年+100店 × 10年 + M&A(W-FK+フレッシュネス265店) + 新フォーマット200店 + 既存閉店▲100店
- **1店SWS**: Slide 03の天井分析に基づく（朝食+デリバリー+DT+価格改善）

### A-2. 国内新業態（チキン・カフェ等）

| 前提 | Base | Upside |
|---|---|---|
| チキン専門店 | 50店 × 1.2億円 = 60億円 | 150店 × 1.5億円 = 225億円 |
| カフェ業態 | 80店 × 1.0億円 = 80億円 | 200店 × 1.2億円 = 240億円 |
| その他新業態 | — | 100店 × 1.0億円 = 100億円 |
| W-FK（2ブランド） | 108店 × 1.5億円 = 162億円 | 200店 × 1.7億円 = 340億円 |
| **合計** | **~300億円** | **~800億円** |

根拠:
- チキン市場 ~1,100億円（KFC独占）にモスチキン認知度を活かして参入
- カフェ市場 ~1.2兆円。モスカフェ既存実績あり
- W-FKはM&A取得後にリブランド・出店加速を想定

### A-3. 海外事業

| 前提 | Base | Upside |
|---|---|---|
| 店舗数 | 450店（微増） | 700店（ASEAN深耕） |
| 1店あたり売上 | 0.67億円 | 0.79億円 |
| **合計** | **300億円** | **550億円** |

根拠:
- Baseは現状の延長（縮小フェーズ後の安定化）
- Upsideはタイ・フィリピンでの出店加速（年+30店）を想定
- 台湾依存度を72%→50%に分散

## 2. B. SC事業（食材サプライヤーとしての外販）

### B-1. 中堅チェーン向けOEM

| 前提 | Base | Upside |
|---|---|---|
| 取引チェーン数 | 20〜30チェーン | 50〜80チェーン |
| 1チェーンあたり供給額 | 10〜15億円/年 | 10〜15億円/年 |
| **合計** | **300億円** | **800億円** |

根拠:
- 50-300店規模の中堅チェーン: 日本国内に約200-300社存在
- ペインポイント（自前CK投資困難、人手不足、品質均一化）はリサーチで確認済
- シコメルフードテックの急成長がニーズの存在を実証
- Baseは市場シェア~10%、Upsideは~25%を想定

### B-2. デリカフーズHD（M&A）

| 前提 | Base | Upside |
|---|---|---|
| 既存売上（M&A時点） | 588億円 | 588億円 |
| 有機成長（年+3%） | +177億円（10年） | — |
| シナジー（モスのネットワーク活用） | — | +200億円 |
| **合計** | **~700億円** | **~1,000億円** |

根拠:
- 既存売上588億円がベースライン（ダウンサイドリスク限定）
- 業務用野菜市場は構造成長中（外食の仕込みアウトソーシング加速）
- モスの契約農家2,379軒との統合で調達力強化

### B-3. 製パン会社 外販分

| 前提 | Base | Upside |
|---|---|---|
| モスグループ内供給 | 自社消費（連結相殺） | 自社消費（連結相殺） |
| **外販売上** | **150億円** | **250億円** |

根拠:
- 第一屋製パン売上272億円のうち、モス向け以外の外販分
- Upsideはモスブランドのバンズ・パン製品の外販拡大

## 3. C. リテール事業

### C-1. 冷凍食品（「モス」ブランド）

| 前提 | Base | Upside |
|---|---|---|
| 市場浸透率 | 冷凍食品市場の0.7% | 冷凍食品市場の2.0% |
| 単価 | 400〜600円/個 | 同 |
| **合計** | **300億円** | **800億円** |

根拠:
- 家庭用冷凍食品市場4,062億円 + 成長率4.4%/年で10年後~6,000億円
- 参照: イートアンドフーズ（大阪王将）食品事業199億円
- モスのブランド認知度は大阪王将を大きく上回る → 同規模以上が射程
- 主力商品: 冷凍モスバーガー、冷凍モスチキン、冷凍ライスバーガー

### C-2. ソース・調味料

| 前提 | Base | Upside |
|---|---|---|
| 市場浸透率 | ソース・調味料市場の0.2% | 0.5% |
| **合計** | **100億円** | **300億円** |

根拠:
- ソース・調味料市場5,770億円（CAGR 6.45%で10年後~10,000億円）
- モスのミートソースは累計13億個販売の実績
- 「モスのソース」としてボトル販売（カゴメのトマトケチャップに類似）

## 5. 感応度分析

### 変数別の影響度

| 変数 | ±1単位の変化 | 全体への影響 |
|---|---|---|
| 国内店舗数 ±100店 | ±156〜195億円 | ±2〜3% |
| 1店あたりSWS ±0.1億円 | ±184〜283億円 | ±3〜5% |
| B2B取引チェーン数 ±10社 | ±100〜150億円 | ±1〜3% |
| 冷凍食品市場浸透率 ±0.5% | ±150〜300億円 | ±2〜5% |
| デリカフーズ成長率 ±2% | ±100〜150億円 | ±1〜2% |

### シナリオ別の合計

| シナリオ | 外食 | SC事業 | リテール | IP | **合計** | **目標比** |
|---|---|---|---|---|---|---|
| **Worst** | 2,500億 | 600億 | 250億 | 50億 | **3,400億** | **31%** |
| **Base** | 3,460億 | 1,150億 | 520億 | 100億 | **5,230億** | **48%** |
| **Upside** | 6,130億 | 2,050億 | 1,300億 | 300億 | **9,780億** | **91%** |
| **Stretch** | 6,500億 | 2,500億 | 1,500億 | 300億 | **10,800億** | **100%** |

### 10,800億円（100%到達）に必要なStretch要素

1. 追加M&A（残余力85億円+新規借入）で食品メーカーを1社追加取得
2. SC事業のB2B外販が想定以上に拡大（空白市場のため上振れ余地大）
3. 海外リテール展開（アジア向け冷凍食品輸出、台湾・ASEAN向け）
4. 業界再編の機会的M&A（JBSのPilgrim's Prideのような破格案件）

## 6. 売上構成の変化

| セグメント | 現状 | 10年後(Base) | 10年後(Upside) |
|---|---|---|---|
| 外食事業 | **100%** | 66% | **63%** |
| SC事業 | 0% | 22% | **21%** |
| リテール | 0% | 10% | **13%** |
| IP | 0% | 2% | **3%** |

→ Upsideでは外食が63%に低下し、SC+リテール+IPで37%を構成。これはDomino's（外食40%:SC60%）に近い多層収益構造。
