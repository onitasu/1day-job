## 5. 主要リスクと対応策（詳細）

### リスク①: M&A統合の失敗

| 影響 | 発生確率 | 対応策 |
|---|---|---|
| SC事業が立ち上がらない | 中 | Phase 1で小型案件（紅梅、35億円）から着手し統合経験を積む |
| | | PMI専任チーム（5〜10名）を組成 |
| | | 紅梅食品は53年の取引関係があり統合リスクが最も低い |

### リスク②: B2B外販の需要不確実性

| 影響 | 発生確率 | 対応策 |
|---|---|---|
| SC事業の売上が計画未達 | 中〜高 | デリカフーズの既存売上588億円がベースラインを保証 |
| | | パイロット取引（3〜5社）で需要検証後に本格展開 |
| | | 「空白市場」のため競合による価格破壊リスクは低い |

### リスク③: リテール事業の立ち上げ遅延

| 影響 | 発生確率 | 対応策 |
|---|---|---|
| 冷凍食品の棚確保が困難 | 高 | まず石井食品の既存チャネルを活用（棚は確保済み） |
| | | EC先行→実績を示してからスーパーに提案 |
| | | 上位5社（ニッスイ/ニチレイ等）がシェア75%の寡占市場だが、「外食チェーン品質」の差別化で棚を獲得 |

### リスク④: 既存事業（外食）の停滞

| 影響 | 発生確率 | 対応策 |
|---|---|---|
| 成長の前提が崩れる | 低〜中 | 外食はBase前提（保守的）で計画。Upsideは上振れ余地として確保 |
| | | ハンバーガー市場は56カ月連続プラス成長中 |
| | | 人口減影響は10年で▲3〜5%程度。客単価上昇で相殺可能 |

### リスク⑤: 財務リスク（のれん・借入負担）

| 影響 | 発生確率 | 対応策 |
|---|---|---|
| M&A後の利益圧迫 | 中 | のれん償却: 4件合計で年15〜25億円（営業利益52億の30〜50%） |
| | | D/E 0.5倍での利息: 年8億円（営業利益の15%） |
| | | **Phase 1の紅梅食品は低リスク（35億円、非上場のためのれん小）** |
| | | 段階的M&Aで各案件のPMI完了を確認してから次に進む |

## 6. 垂直統合の5つの法則（グローバル事例から）

| 法則 | 内容 | モスへの適用 |
|---|---|---|
| **法則1** | 「単一→マルチ」で売上3-5倍 | パティ→バンズ→野菜→加工食品 |
| **法則2** | 高マージン事業は「最後に」取りに行く | Phase 1-2は規模構築、Phase 3でリテール（高マージン）をスケール |
| **法則3** | 危機時のバーゲン買収がIRR最高 | W-FK（PEファンドのイグジット案件）はバーゲン機会 |
| **法則4** | 卸売は低マージンだが、PBで改善 | デリカフーズの外販にモスPBを導入 |
| **法則5** | 海外進出はM&Aでしか成功しない | Phase 3で海外食品メーカーの買収を検討 |

## 7. エグゼクティブサマリー（再掲）

1. **問い**: モスは10年でマクドナルドに並べるか
2. **現状**: 売上1,302億 vs 目標10,800億。ギャップ8.3倍
3. **課題**: 既存モデルの天井は最大6,420億（目標の59%）。店舗ビジネスだけでは届かない
4. **戦略方向**: バリューチェーン拡張。「食材卸売会社」の実態を活かし「食のインフラ企業」へ転換
5. **具体策**: 4セグメント（自社FC/中堅チェーンB2B/リテール/野菜卸売）に向けて段階的M&A（~315億、4件）で製造・供給能力を獲得
6. **到達見込み**: Base 5,230億（48%）、Upside 9,780億（91%）。10,800億にはストレッチ要素が必要だが、8,000〜10,000億で「実質対等ポジション」を確立
7. **最初の一手**: 紅梅食品工業の完全子会社化（~35億、投資余力の9%）。リスク最小・効果最大の起点
