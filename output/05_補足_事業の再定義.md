# 補足資料 05：事業の再定義 — 構造的教訓とモスの未活用資産

## 1. Tysonの法則 — 詳細データ

### Tyson Foods 売上推移（養鶏$2.5B → $54B）

| 年 | 売上 | イベント |
|---|---|---|
| 1989 | $2.54B | Holly Farms買収、養鶏No.1に |
| 1999 | $7.36B | IBP買収前 |
| **2001** | **$23B** | **IBP買収（$4.4B）→ 牛肉・豚肉参入、売上3.1x** |
| 2014 | $37.6B | Hillshire買収前 |
| **2015** | **$41.4B** | **Hillshire Brands買収（$8.55B）→ 加工食品参入** |
| 2025 | **$54.4B** | 現在 |

### セグメント別構成（FY2024）

| セグメント | 売上 | 構成比 | 営業利益率 |
|---|---|---|---|
| Beef（牛肉） | $20.5B | 38.4% | **-1.9%（赤字）** |
| Chicken（鶏肉） | $16.4B | 30.8% | +6.0% |
| **Prepared Foods（加工食品）** | **$9.9B** | **18.5%** | **+8.9%** |
| Pork（豚肉） | $5.9B | 11.1% | -0.7% |

### Tysonの法則の本質

**川上素材（パティ・野菜）で「規模」を作り、加工食品（冷凍食品・ソース）で「利益」を取る。**

- 牛肉事業は売上最大（$20.5B）だが赤字。規模を維持するための「土台」
- 加工食品は売上18.5%だが、利益率8.9%で**利益の大部分を創出**
- Jimmy Dean、Ball Parkなどの「ブランド」が価格決定力を生む
- **ブランド付き加工食品 = 利益エンジン**

### モスへのインプリケーション

| Tyson | モスに置き換えると |
|---|---|
| 養鶏（川上素材） | パティ製造、野菜調達（規模を作る） |
| 牛肉・豚肉（M&Aで規模拡大） | デリカフーズHD、製パン会社（SC規模を拡大） |
| **Prepared Foods（利益エンジン）** | **「モス」ブランド冷凍食品・ソース（利益を取る）** |

## 2. カゴメの教訓 — 詳細データ

### カゴメの売上推移

| 年度 | 売上高 | 備考 |
|---|---|---|
| 2016/12期 | 2,025億円 | |
| 2022/12期 | 2,056億円 | |
| **2024/12期** | **3,069億円** | **Ingomar買収で+36.5%** |

### 事業構成

| セグメント | 売上規模 | 構成比 |
|---|---|---|
| 国内加工食品 | ~2,000億円 | ~65% |
| **国内農事業** | **~100億円** | **~3%** |
| 国際事業 | 急拡大中 | ~30%+ |

### カゴメの教訓の核心

**農業は「ブランドの裏付け」に留め、利益は加工食品で稼ぐ。**

- 自社農場産トマト: 約1万トン（日本の消費量70万トンの約1.5%）
- 原料の大半は契約栽培+海外調達
- 農事業は連結の3%。**利益の97%は加工食品事業**
- 農業は「カゴメ=トマトの会社」というブランドイメージの裏付けとして不可欠だが、利益の源泉ではない

### キユーピーモデル — 「業務用+家庭用」の二刀流

| セグメント | 売上 | 構成比 | 営業利益前年比 |
|---|---|---|---|
| 市販用（家庭向け） | 1,774億円 | 36.9% | +43.6% |
| **業務用（外食・中食向け）** | **1,653億円** | **36.2%** | **+189%** |
| 海外 | 783億円 | 19.5% | +39.4% |

→ 市販用と業務用がほぼ1:1。**B2B+B2Cの両面展開**が需要安定性と価格交渉力を両立。

## 3. モスファームの失敗 — 何が問題だったか

### 時系列

| 年 | 出来事 |
|---|---|
| 1997 | 「作り手の顔が見える安全な野菜」を宣言 |
| 2006 | 第1号モスファーム（静岡）設立 |
| 2013-2017 | 熊本、北海道、千葉等に展開 |
| **2020年6月** | **広島支部を最後に新規展開を休止** |

### 失敗の構造

| 問題 | 詳細 |
|---|---|
| 収益化困難 | 「蓋を開けてみるとそんなにうまく事業は成り立たなかった」（モス幹部） |
| 規模の限界 | 全10拠点で使用量の**約1割**のみ。残り9割は外部調達 |
| 規制の壁 | 農地法の制約で議決権は4分の1以下に抑制が必要 |
| 天候リスク | 農業固有の不確実性 |
| 人材不足 | 農業専門人材の確保が困難 |

### 日本の農業法人の実態

| 指標 | データ |
|---|---|
| 法人数 | 約33,000法人（2023年） |
| 平均売上高 | 3億9,214万円 |
| 売上10億円超の法人 | 極めて限られる |
| 農業総産出額 | 10兆7,801億円（2024年） |

→ **農業は個別法人の規模が小さく、収益化が構造的に困難**。カゴメですら農業は連結の3%。モスが農業を利益の源泉にするのは非現実的。

## 5. バリューチェーン各層の収益化ポテンシャル

```
                  現在の収益化         転換後の収益化

【川上】農業         ×（コストセンター）    △（ブランド裏付け、維持）
【中流】製造         ×（外部に流出）        ◎（紅梅食品子会社化で内製）
【中流】物流         ×（3PLに支払い）       ○（デリカフーズの物流活用）
【川下】FC卸売       ◎（491億円）          ◎（内製化でマージン拡大）
【川下】店舗         ◎（SWS1,302億円）     ◎（既存+新業態）
【新規】B2B外販      ×（ゼロ）             ◎（中堅チェーン向けOEM）
【新規】リテール      ×（EC小規模のみ）      ◎（冷凍食品・ソース商品化）
```

→ 7つの層のうち、現在収益化しているのは**2つ（FC卸売+店舗）のみ**。残り5つに収益化の余地がある。
