# Slide 04：天井の構造的原因

## メインメッセージ
> **モスはバリューチェーン上に複数の資産を持ちながら、「FC店への食材卸売」の1層でしか収益化していない。この単層構造が売上の天井を規定している。そして、この構造的制約の裏返しこそが最大の成長機会である。**

---

## 事実：モスは「食材卸売会社」である

連結売上961億円の過半は、FC店への食材・資材の卸売で構成される。

| 項目 | 金額 | 構成比 |
|---|---|---|
| **FC店への食材・資材卸売** | **491億円** | **51%** |
| 直営店売上 | 241億円 | 25% |
| 海外事業 | 166億円 | 17% |
| ロイヤリティ等 | 29億円 | 3% |
| その他 | 34億円 | 4% |

東証の業種分類が「卸売業」である理由。**売上の過半は食材の卸売**。

## 構造的矛盾：食材卸売会社なのに、食材を製造していない

```
パティ・ソース → 紅梅食品工業（22%出資の外部企業）に委託
バンズ        → 山崎製パンに委託
物流          → 外部3PLに全面委託
```

→ **製造マージンが外部に流出する「ファブレスモデル」**。Domino'sも同様にFC店に食材を供給するが、**自社SCで製造・配送まで行い、売上の60%をSC事業で計上**している。モスはその構造を持ちながら、製造を外部に渡している。

## 天井の正体：1層しか収益化していない

| モスが持つ資産 | バリューチェーン上の位置 | 収益化 |
|---|---|---|
| モスファーム10拠点、契約農家2,379軒 | 川上（農業・調達） | × |
| 紅梅食品工業22%出資、53年の取引 | 中流（製造） | × **マージン外部流出** |
| 13配送センター、物流網 | 中流（物流） | × |
| **FC網1,307店（卸売491億円）** | **川下（卸売）** | **◎ ← ここだけ** |
| ブランドIP「安心・品質・国産」 | 全層横断 | × 店舗集客のみ |

**→ 5つの資産のうち収益化は1つだけ。「店舗数 × 客単価」の掛け算しか持たないから、天井がある。**

## なぜマルチブランド化・高単価化では天井を破れないか

Slide 03で検証した通り、マルチブランド化（W-FK買収等）や高単価化を織り込んでも天井は62%に留まる。これは**ポートフォリオの中身を変えているだけで、収益化の「層」は増えていない**から。

```
【マルチブランド化】 同じ1層の中での最適化
  モスバーガー（客単価800円）  ─┐
  W-FK（客単価1,000円）        ├→ すべて「店舗 × 客単価」の1層
  プレミアム業態（客単価1,500円）─┘

【バリューチェーン拡張】 層そのものを増やす
  製造（内製化マージン）  ─→ 新たな層
  B2B外販（OEM供給）     ─→ 新たな層
  リテール（冷凍食品）    ─→ 新たな層
```

→ **天井を突破するには、掛け算の変数を増やす（=収益化の層を増やす）必要がある。**

## 制約の裏返し＝機会

この構造的制約は、そのまま裏返すと成長機会になる：

- 製造を内製化 → **製造マージンをグループ内に取り込める**
- 余剰キャパシティを外販 → **新たな顧客層への売上が生まれる**
- ブランドIPを商品化 → **リテール事業が立ち上がる**
- 農業基盤をSCに統合 → **差別化された調達力になる**

**→ 「1層の収益化」を「多層の収益化」に変えることで、店舗数×客単価の天井を突破できる。**
