# Slide 09：実行ロードマップと最初の一手

## メインメッセージ
> **3フェーズで段階的に実行。最初の一手は紅梅食品工業の完全子会社化（~35億円、投資余力の9%）。53年の取引関係があり統合リスク最小、製造マージン取込の効果は即時。**

---

## 3フェーズの設計

| | Phase 1（0〜3年） | Phase 2（3〜7年） | Phase 3（7〜10年） |
|---|---|---|---|
| **テーマ** | **基盤構築** | **事業拡大** | **スケール** |
| **売上目標** | ~2,500億 | ~5,000〜6,000億 | ~8,000〜10,000億 |
| **M&A** | 紅梅食品（~35億）、W-FK（~100億）、デリカフーズHD（~130億） | 製パン会社（~70億）、石井食品（~80億） | 追加M&A検討 |
| **外食** | 既存レバー実行（朝食・デリバリー・DT）、FC出店加速 | 新業態立ち上げ（チキン・カフェ） | 新業態のスケール |
| **SC事業** | 製造能力把握、デリカフーズ統合、B2B営業体制構築 | 中堅チェーンへOEM開始、外販比率拡大 | SC事業の本格スケール |
| **リテール** | 冷凍食品の商品開発・テスト販売 | スーパー・コンビニへ本格展開、ソース商品化 | リテール事業スケール、海外輸出 |
| **投資額** | ~265億 | ~150億 | 状況に応じて |

## 主要リスクと対応策

| リスク | 影響 | 対応策 |
|---|---|---|
| M&A統合の失敗 | SC事業が立ち上がらない | Phase 1で小型案件（紅梅/35億）から着手し統合経験を積む |
| B2B外販の需要不確実性 | SC事業の売上未達 | デリカフーズの既存売上588億がベースラインを保証 |
| リテールの棚確保困難 | 冷凍食品が売れない | 石井食品の既存チャネルを活用。EC先行で実績を作る |
| 既存外食事業の停滞 | 前提が崩れる | 外食はBase前提で計画。Upsideは上振れ余地 |
